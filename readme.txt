A Simple plugin allow you to upload category image and those displayed in front end with diff. display options.

== Description ==

Category Views is a simple plugin allow you to upload category image with following widget options :

* Number of categories to display
* Ordering
* How to show categories
* Show empty categories?
* Show posts count?			


== Installation ==
1. Unzip 'webline-category-views.zip' to the '/wp-content/plugins/' directory or add it by 'Add new' in the Plugins menu.
2. Activate the plugin through the 'Plugins' menu.
3. Done.
